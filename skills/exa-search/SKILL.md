# Exa Search

This skill provides access to Exa's neural search engine and research tools.

## Tools
- `web_search_exa`: Neural search for high-quality links.
- `web_search_advanced_exa`: Advanced search with filters.
- `deep_search_exa`: Multi-step neural search.
- `deep_researcher_start`: Begin a long-running research task.
- `deep_researcher_check`: Check status/results of deep research.
- `crawling_exa`: Extract clean markdown from URLs.
- `get_code_context_exa`: Search for code and documentation.
- `company_research_exa`: Find company information.
- `linkedin_search_exa`: Search LinkedIn profiles.

## Usage
Use Exa when standard search fails to find specific worldbuilding nuances. It works best with descriptive, "link-like" phrases.
